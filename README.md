# Chinook SQL Business Analytics Project

## Overview
This project is a SQL-based business analytics case study using the Chinook Database. It demonstrates how structured data can be analyzed using SQL to generate meaningful business insights.
The analysis was performed using MySQL Workbench, focusing on customer behavior, revenue trends, regional performance, and advanced SQL techniques.

---

## Project Objectives
- Perform exploratory data analysis on customer and sales data
- Analyze revenue performance across regions
- Use SQL joins to understand relational data structure
- Apply subqueries and window functions for advanced analytics
- Generate business insights from raw transactional data
- Improve query performance using indexing techniques

## 📊 Key Insights Generated
1. The USA generated the highest revenue, making it the strongest market for the business.
2. Customer ranking analysis identified the highest-spending customers contributing most to total revenue.
3. Helena Holý emerged as the top-paying customer, highlighting key customers for retention strategies.
4. Best-selling tracks analysis revealed the most popular music products driving sales performance.
5. Customer purchase frequency indicated strong customer engagement and repeat buying behavior.
